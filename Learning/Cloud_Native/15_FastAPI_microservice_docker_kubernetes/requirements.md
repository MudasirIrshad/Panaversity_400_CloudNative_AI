# Requirements for Two FastAPI Applications

## Overview
Create two separate FastAPI applications using `uv` as the package manager, with both apps utilizing in-memory data storage.

## Application 1
- Minimal FastAPI app
- In-memory data storage
- Basic CRUD operations

## Application 2
- Minimal FastAPI app
- In-memory data storage
- Basic CRUD operations

## Package Management
- Use `uv` for package management
- Generate requirements.txt files for dependencies

## Data Storage
- Both applications must use in-memory storage only
- No external databases or persistent storage
